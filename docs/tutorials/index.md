# Tutorials

Tutorials are lessons. They take you by the hand through a complete, working experience so
you learn how Katalyst fits together. Follow them start to finish — they are written to
work every step of the way.

- **[Getting started](../getting-started.md)** — Build your first Katalyst service from an
  empty project to a running, database-backed HTTP endpoint. Start here if you are new.

Once you can build and run a service, move on to the [how-to guides](../how-to/index.md)
for task-focused recipes, or the [reference](../reference/index.md) to look up specifics.
